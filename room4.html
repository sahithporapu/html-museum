📋 Tables in HTML

The <table> tag is used to display data in rows and columns, just like in Excel.

A table is made of:

  • <tr> Table Row
  • <td> Table Data (cell)
  • <th> Table Header (bold)

📊 Rowspan and Colspan in Tables

In HTML tables, colspan and rowspan are attributes used to merge cells:

  • colspan → Combines multiple columns into one cell (horizontal merge)
  • rowspan → Combines multiple rows into one cell (vertical merge)
