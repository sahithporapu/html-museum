The <iframe> tag in HTML is used to embed another webpage or resource **within your current webpage**. It creates a "window" inside the page that can display videos, maps, websites, or documents from another source.

üîç What Does iframe Stand For?

iframe stands for inline frame. It acts like a mini browser window inside your page.

üß© Basic Syntax

<iframe src="https://www.example.com" width="400" height="300"></iframe>

üìå Common Attributes:

  • src URL of the page to embed
  • width Width of the frame (in pixels or %)
  • height Height of the frame
  • title Accessible name for screen readers
  • allowfullscreen Lets embedded video go fullscreen (for YouTube, etc.)
