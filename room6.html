🧾 HTML Form Elements

📌 <form>

The <form> tag is used to create a section that collects user input. It can include fields like name, email, and feedback. When the form is submitted, the data is sent to a server or processed by the browser.

🏷 <label>

The <label> tag gives a name or description to an input field. It improves accessibility and allows users to click the label to focus the input. It connects to an input using the for attribute.

✍️ <input>

The <input> tag is used to collect user information. It supports various types such as text, number, email, password, checkbox, and more.

📦 <fieldset>

The <fieldset> tag is used to group related form elements together. It draws a box around the group, making the form more organized and readable.

🏷 <legend>

The <legend> tag is used inside a <fieldset> to give a title or heading to the grouped section. It helps users understand what that section of the form is about.
