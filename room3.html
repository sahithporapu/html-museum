🏛 Multimedia in HTML

HTML allows us to add multimedia elements like images, audio, and video to make webpages more engaging.

📸 Image

The <img> tag is used to display images.
<img src="museum.jpg" width="300" alt="Museum Image">

🎧 Audio

The <audio> tag is used to add sound or music.
<audio controls><source src="sound.mp3"></audio>

🎬 Video

The <video> tag is used to display videos.
<video width="320" height="240" controls><source src="clip.mp4"></video>
