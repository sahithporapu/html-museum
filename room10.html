📋 What Are <meta> Tags?

The <meta> tag is used inside the <head> section of an HTML document to provide **metadata** — information about the webpage that isn't displayed on the page itself.

Meta tags help browsers, search engines, and social media platforms understand how to handle your page.

🔹 Common Uses of Meta Tags

  • 🌐 Define the character encoding (e.g., UTF-8)
  • 🔍 Add descriptions for search engines (SEO)
  • 📱 Make your website responsive on mobile devices
  • 🚫 Control how search engines index your site

📌 Common Meta Tags

<meta charset="UTF-8">
Defines character encoding (recommended for all HTML pages)

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Makes the website responsive on mobile devices

<meta name="description" content="Learn HTML in an interactive museum-style format.">
Describes your webpage for search engines (SEO)

<meta name="author" content="Sahith">
Declares the author of the webpage

<meta http-equiv="refresh" content="10">
Refreshes the page every 10 seconds

🧪 Where Do Meta Tags Go?

Meta tags are always placed inside the <head> section of your HTML document: 

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Museum</title>
  </head>
  <body>
    ...
  </body>
</html>

💡 Why Are Meta Tags Important?

  • ✅ Improve search engine rankings (SEO)
  • 📱 Make your website mobile-friendly
  • 🛠 Help browsers display the page correctly
  • 🎯 Provide useful info like author or refresh rules
